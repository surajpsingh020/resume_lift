🚀 AI Resume Builder v2.0 - Docker Edition

Major Features:

✨ AI-Powered Content Generation
- Smart education descriptions using Google Gemini 2.5 Flash
- Keyword extraction and context-aware summaries
- One-click AI generation with loading states

💾 Enhanced User Experience
- Auto-save every 2 seconds - never lose your work!
- Export to Word (.docx) with professional formatting
- Keyboard shortcuts (Ctrl+← / Ctrl+→) for form navigation
- Real-time preview updates
- Input validation for email and phone numbers
- Loading skeletons for better UX

🎨 Design Improvements
- 21 professional theme colors - choose from curated palette
- Improved toast notifications with icons and better positioning
- Fixed double preview bug on Skills page
- Responsive design across all devices

🐳 Complete Docker Support
- Production-ready multi-stage builds (Frontend: ~50MB, Backend: ~150MB)
- Development environment with hot-reload enabled
- One-click startup scripts: docker-start-prod.bat, docker-start-dev.bat
- Health checks for all services
- Security best practices: Non-root user, security headers, secrets management
- Nginx configuration: Gzip compression, SPA routing, 1-year asset caching

📦 Quick Start

Docker (Recommended):
```
git clone https://github.com/surajpsingh020/resume_lift.git
cd resume_lift
docker-start-prod.bat

Frontend: http://localhost:8080
Backend: http://localhost:5001
```

Local Development:
```
start-all.bat
```

🛠️ Tech Stack
- Frontend: React 18.3, Vite 5.3, Redux Toolkit, Tailwind CSS, Shadcn/ui
- Backend: Node.js v22, Express 4.19, Mongoose 8.5, JWT auth
- AI: Google Generative AI (Gemini 2.5 Flash)
- Database: MongoDB Atlas (cloud)
- Docker: node:20-alpine, nginx:alpine

🔧 Configuration

Backend .env (required):
MONGODB_URI=your_mongodb_connection_string_here
JWT_SECRET=your_jwt_secret_key_here
GEMINI_API_KEY=your_gemini_api_key_here
ALLOWED_SITE=http://localhost:8080
NODE_ENV=Dev

Frontend .env.local:
VITE_APP_URL=http://localhost:5001/
VITE_GEMINI_MODEL=models/gemini-2.5-flash
